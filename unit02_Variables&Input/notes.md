A variable stores a value in memory

Basic types:
- int       //whole numbers
- float     // decimal numbers
- char      // single character

Example:

int age;
float height;

Input with scanf
scanf("format", &variable);

Format specifiers:
- %d    int
- %f    float
- %c    char

Example
int age;
scanf("%d", &age);

(&= address)

/////////////////////////////////////////////////////////////////////////

Variables:
- Must be declared before use
- int stores integers

Input:
- scanf reads user input
- &variable passes address
- %d used for int input
