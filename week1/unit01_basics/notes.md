UNIT 01- BASICS & OUTPUT

C Program structure
- Every C program starts in main()
- main must return int
- return 0; means successful execution


Header files
- #include <stdio.h> enables input/output
- Needs for printf

Output with printf
- printf prints text to the terminal 
- \n creates a new line

Format specifiers
- %d   integer
- %f   float
- %.2f float with 2 decimals

Running a C program
1. Compile: gcc file.c -o program
2. Run: ./program
- .c files cannot be executed directly

Notes
- C is a compiled language
- Program must be recompiled after changes
