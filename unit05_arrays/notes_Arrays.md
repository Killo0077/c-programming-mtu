The “formula” also has a name

This loop:

```python
for (int i = 0; i < size; i++) {
    // array[i]
}
```


# Is often called:

**standard array traversal loop

canonical for loop for arrays

idiomatic C array loop**


# “Idiomatic” means:

the normal, expected way people do it

Why everyone memorizes it (important reassurance)



# Arrays are:

zero-indexed

contiguous in memory

accessed by index

So the language itself forces this pattern.
